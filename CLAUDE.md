# clay-events-system-JLD

## STACK fijo, ya está decidido y no se puede cambiar

- Backend: node.js + TypeScript + Express
- Cola: Redis
- BD: MongoDB
- Worker: Proceso node separado del servidor HTTP
- FrontEnd: React + TypeScript 
- Test: Jest
- Entorno: Docker Compose
